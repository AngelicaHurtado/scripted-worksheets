### Worksheet Instructions

1. Click on "js-worksheet.md" to open it.

2. Press 'Fork' at the top right

2. Create a branch with your name (top left).

3. Click "js-worksheet.md" again. 

3. Click the pencil (top right) to edit and type in your responses.

4. Commit changes to save your work. You can always go back to it and edit it.

####When you're ready to submit your work:

5. Go back to the main worksheet folder and click on the big green button that says 'Compare & Pull Request'.

6. Leave a commit message in the body and tag your instructor using their GitHub username, i.e. @mackenco

7. Click on 'Create pull request' at the bottom.

8. You're done! You''ll be redirected to page that shows you your pull request.
